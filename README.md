# Countdown
Create count down to an event
Inspired by ZUI days mobile app

## Set up instructions
To get started, clone the repository and install the dependencies:

<code> 
git clone https://github.com/heyjiawei/Countdown.git
</code>

Then, 
<code>
npm install
</code>

Followed by 
<code>
npm run build
</code>

and 
<code>
npm start
</code>
